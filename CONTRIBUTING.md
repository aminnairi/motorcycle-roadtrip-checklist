# Contributing

## Requirements

- [Git](https://git-scm.com/)
- [Node](https://nodejs.org/en)
- [NPM](https://www.npmjs.com/)

## Clone

```bash
git clone git@github.com:aminnairi/motorcycle-roadtrip-checklist
```

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
